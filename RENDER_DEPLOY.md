# Render Deployment Guide: Faculty Finder (100% Free)

## Step 1: Push to GitHub
If you haven't already, create a private/public repo and push your code.

## Step 2: Create Web Service
1. Go to [Render.com](https://render.com).
2. New -> Web Service -> Connect your Repo.
3. Select **Docker** as the Runtime.
4. Choose the **Free** instance type.

## Step 3: Deploy
- Render will use the `Dockerfile` to build and serve your app at a public URL.

**Tip:** After deployment, visit your URL to see the AI Dashboard live!
